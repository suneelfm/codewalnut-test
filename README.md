## **Project Setup**

1. Clone the repository:

   ```bash
   git clone https://github.com/suneelfm/codewalnut-test.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Run tests:
   ```bash
   npm vitest
   ```
